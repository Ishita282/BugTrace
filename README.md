# BUGTRACE - BUG TRACKING SYSTEM

BugTrace is a full-stack web application built to manage and track software bugs efficiently. It allows users to create, update, and delete bugs while maintaining their status in an organized dashboard.

---

## FEATURES
    - User Authentication (Login/Register)
    - Create, Update, Delete Bugs
    - Bug Status Tracking (Open / In Progress / Closed)
    - Responsive Dashboard UI
    - Clean and simple interface for issue management

---

## SETUP INSTRUCTIONS

1. **Clone the repository**:
```bash
git clone https://github.com/Ishita282/BugTrace.git
cd BugTrace
```

2. **Setup Backend**:
```bash
cd server
npm install
npm start
```

3. **Setup Frontend**:
```bash
cd client
npm install
npm start
```

4. **ENVIRONMENT VARIABLES (.env)**:
```bash
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
